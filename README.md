## 功能列表
开源论坛、问答系统，现有功能多社交平台登录(Github，Gitee)提问、回复、通知、最新问答、最热热大、消除零回复等功能。

## 本地运行手册
1. 安装必备工具  
   JDK，Maven
2. 克隆代码到本地
```sh
git clone https://github.com/seeingAndThinking/community.git
```
